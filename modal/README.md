## 기능
- [x] 버튼 클릭시 레이어팝업 열리고, 팝업내의 닫기버튼 클릭시 레이어팝업 닫힘 
- [x] 화면의 중앙에 레이어팝업 위치, dimmed(반투명배경) 처리
- [x] 레이어팝업 열고 닫힐때 애니메이션 추가
- [x] 레이어팝업 열였을때, body의 스크롤 제어
- [x] 레이어팝업 이외의 다른곳(dimmed)을 클릭시, 닫힘
- [ ] 다양한 상황 대처
