# IF 조건문

## 자바스크립트
- [결과화면 보기](https://yeony1011.github.io/2019script_ex/if/if.html)

## 기능
1. [x] 입력받은 숫자가 홀수인지 짝수인지 알려주는 프로그램 작성   
2. [x] 입력받은 문자가 대문자로 되어 있으면 '대문자 문장입니다.', 소문자로 되어 있으면 '소문자 문장입니다.', 그 외에는 '보통 문장입니다.' 라고 출력하는 프로그램 작성
